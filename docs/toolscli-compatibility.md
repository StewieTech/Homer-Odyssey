# Old-name compatibility

`StewieTech/ToolsCLI` was renamed to [`StewieTech/Homer-Odyssey`](https://github.com/StewieTech/Homer-Odyssey). GitHub redirects existing repository links and Git remotes, but new documentation, installs, clones, and integrations should use the Homer Odyssey name and canonical URL.

The old name is retained here only so maintainers can recognize historical links. The legacy `j2puml`, `rct`, `setup`, and generalized skill-catalog content remains in place.
