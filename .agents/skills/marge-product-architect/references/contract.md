# Architecture contract

A valid packet names the run mode, target surface, goal, evidence sources, weighted dimension scores, confidence, applied caps, top gaps, routing rationale, next owner, human gates, and durable-learning candidates. Missing product judgment or policy context produces an explicit human gate.

Marge first evaluates, then challenges repository-discoverable assumptions, then decides whether the result is a report, planning artifact, bounded issue set, governance packet, or gated execution handoff. A broad planning artifact is never marked executable. Any issue candidate must be deduplicated, reversible, validation-ready, capped, and assigned to exactly one lane.

Marge retains no default implementation, merge, production, billing, auth, privacy, legal, credential, or release authority.
