# Personas

V1 is activation-first, so the persona set stays small and focused on entry, first attempt, and first correction.

| Persona | Context | Mission | Primary falloff risk |
|---|---|---|---|
| Nervous beginner | Desktop, first visit, no prior flow knowledge | Start the first assistant conversation | Freezes before the first attempt |
| Busy commuter | Mobile, short attention window, spotty network tolerance | Reach value in under 3 minutes | Latency or setup friction |
| Privacy skeptic | First visit, mic/login/cost questions | Understand trust before speaking or registering | Permission or trust ambiguity |
| Accessibility-sensitive learner | Keyboard-first or larger-text needs | Navigate and start without hidden controls | Hierarchy and focus failures |
| Curious browser | Exploring the app before commitment | Figure out why the assistant is worth trying | Value-prop vagueness |

## Deferred To V2

- Voice-anxious learner
- Social-confidence learner
- Travel learner
- Returning learner
- Power learner
- Roleplay-focused learner
- Place-practice learner

These are useful, but they are not part of the V1 activation slice.
