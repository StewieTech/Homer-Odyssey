# Analysis contract

## Required sequence

1. Ingest one evidence-backed finding or direct QA claim.
2. Reproduce the smallest end-to-end path once and record the environment.
3. If the result is ambiguous, map callers, boundaries, and state ownership before deeper diagnosis.
4. Score reproduction confidence, evidence quality, user impact, actionability, and lane fit.
5. Search the target's issues, changes, and durable registers for equivalent active work.
6. Shape one bounded packet or return an evidence-lock no-op.
7. Route to execution readiness, deeper planning, or governance without implementing.

The packet includes reproduction steps, content-minimized evidence references, impact, severity, confidence, dedupe result, first slice, non-goals, validation, unresolved ambiguity, and exactly one recommended lane. Policy, trust, privacy, billing, entitlement, legal, safety, or production ambiguity routes to governance.
